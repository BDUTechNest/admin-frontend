## Nextjs Admin Frontend App


## Development
To start bulding follow these steps:
1. Clone this repo.
2. The project uses pnpm to handle package management. Run the following command to install dependencies:
```bash
pnpm install
```
3. Start the dev server:
```bash
pnpm dev
```
